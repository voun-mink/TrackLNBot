# Телеграм-бот для ведения списков

### Бот разработан на NodeJS, PostgreSQL и n8n

## Описание

Бот создан как проект, для изучения возможностей платформы n8n и не несёт целей рименения в повседневной жизни.

## Используемые технологии

В ввиду требования платформой n8n домена для сканирования бота Телеграмм, на наличие изменений, был созан небольшой обработчик активности, при использовании библиотеки node-telegram-bot-api.
